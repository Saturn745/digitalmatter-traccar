# digitalmatter-traccar
Simple program that acts as a third-party DigitalMatter tracker TCP server and forwards all data to a Traccar server. 

# Relevant Info
https://support.digitalmatter.com/integration/setting-up-direct-to-3rd-party-server-messaging-on-devices
